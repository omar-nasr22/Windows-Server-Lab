# Windows Server Lab

This project demonstrates a clean and practical Windows Server environment built for learning and hands-on practice.

## Overview
The lab simulates a small enterprise network using Windows Server and a domain-joined client machine.  
It focuses on core system administration tasks commonly required for IT System Administrator and Help Desk roles.

## Technologies Used
- Windows Server
- Active Directory Domain Services (AD DS)
- DNS
- DHCP
- Group Policy
- Windows Client (Domain Joined)

## Lab Implementation
- Configured a Windows Server as a Domain Controller.
- Designed an Active Directory structure using Organizational Units.
- Created and managed domain users and security groups.
- Implemented DHCP for automatic IP address assignment.
- Joined client machines to the domain.
- Applied Group Policy for security restrictions and user environment configuration.
- Mapped a network drive for IT users using Group Policy Preferences.

## Project Purpose
This lab was created to strengthen practical understanding of Windows Server administration and to simulate real-world enterprise scenarios.
